# UC admissions & the LCFF+ poverty threshold — interactive

Open `index.html` in any browser (works on phones), or serve with GitHub Pages. Self-contained: data embedded, no dependencies.

Each dot is a California high school (pooled 2021-2025 UC applicants, >=20). The vertical line is the LCFF+ cutoff at UPP=75%; the two trend lines are the local-linear regression-discontinuity fit on each side, and the green bracket + top label give the estimated jump. Toggle San Diego (~+6 pt jump) vs Berkeley (no jump).

Controls: scroll or pinch to zoom, drag to pan, tap a dot for details; a checkbox toggles dot sizing by applicants.

Method & caveats: `../LCFF Threshold RD 2026-07-06/READOUT.md`. Not a claim about intent; local to the cutoff.
